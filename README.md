# ACEest Fitness & Gym – CI/CD DevOps Project

## Project Overview
This project demonstrates implementation of a CI/CD pipeline for a fitness management application using modern DevOps practices.

The application evolved through multiple versions (v1 → v3.2.4) showing progressive feature enhancements such as database integration, analytics, and reporting.

The goal of this assignment is to demonstrate:
- Version control using Git & GitHub
- Unit testing using Pytest
- Containerization using Docker
- Continuous Integration using GitHub Actions
- Build validation using Jenkins

---

## Project Structure

app.py – main application  
requirements.txt – dependencies  
test_app.py – unit tests  
Dockerfile – container configuration  
.github/workflows/main.yml – CI pipeline configuration
